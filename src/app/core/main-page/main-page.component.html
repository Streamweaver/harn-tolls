Tenant Generator

A tenant household generator for Harn Manor. This creates the base population of tenant households on the manor. It includes generation of their social class, any crafting specialties, number of people in the household, and acres of land managed. It also provides some information on Rent and Fees depending on the values assessed under the manor.

Harn Roll20 Character Sheet

I created a Harn character sheet for use in the Roll20 virtual tabletop system. It features a number of auto-calculations and provides macros for rolls that include penalties. The sheet is not currently included in the roll20 character sheet list but has been submitted. To use it you'll have to have a pro account and select a custom sheet for your game and copy in the code from the html and css files in the github repo.
